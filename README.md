# Mí-á-Chài Taiwan Market Website

Static GitHub Pages site for the Mí-á-Chài Taiwan Market iOS app.

Production URLs after GitHub Pages deployment:

- Privacy Policy: `https://twbraam.github.io/miaachaitaiwan/privacy/`
- Support: `https://twbraam.github.io/miaachaitaiwan/support/`

The site is dependency-free HTML and CSS. To preview locally:

```sh
python3 -m http.server 8080
```

Deployment is handled by `.github/workflows/pages.yml` on pushes to `main`.
